\begin{block}{项目概述与动机}
\normalsize
大语言模型已无处不在，但 ``身体'' 仍是缺失的最后一公里。本项目让学生在
\textbf{不到一节课}的时间里，亲手感受\textbf{经典控制}与\textbf{大模型驱动}
在同一物理世界中的分野——全程跑在一台普通 MacBook 上，无需 GPU 与云端。
\end{block}

\vspace{0.6cm}

\begin{block}{三种交互模式}
\normalsize
\begin{itemize}\setlength\itemsep{8pt}
  \item \textbf{接球 · 单步闭式弹道反解}（经典控制，无 AI）——每帧重拟合弹道、
        闭式 IK 解算；自动基准 \textbf{92--100\%} 命中（固定种子可复现）。
  \item \textbf{自然语言驱动 · VLA}——Claude 把 ``pizza tower'' 解析成结构化动作。
        定位为 \textbf{LLM 意图解析器}（L$\to$A），非 RT-2 式端到端 VLA。
  \item \textbf{装饰手势}——wave / point / bow / dance，赋予机器人``意图感''。
\end{itemize}
\end{block}

  \item generation counter 防止旧线程覆盖新命令。
\end{itemize}
