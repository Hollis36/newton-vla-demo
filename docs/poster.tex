\begin{secbox}{引言与动机}
\normalsize
大语言模型已能写代码、规划、对话，但``身体''仍是缺失的一环。课堂场景中学生难以直观区分
\textbf{经典控制}与\textbf{学习驱动}，亦难以亲手验证大模型在闭环物理中的行为。本系统将二者
并置于同一可交互画面，降低具身智能教学的硬件与认知门槛。
\end{secbox}

\vspace{0.5cm}

\begin{secbox}{三种交互模式}
\normalsize
\begin{itemize}\setlength\itemsep{5pt}
  \item \textbf{接球}——单步闭式弹道反解（经典控制，无 AI）；固定种子自动基准命中 92--100\%。
  \item \textbf{自然语言驱动}——\textbf{LLM 意图解析器}（L$\to$A，无像素输入），区别于 RT-2 式端到端 VLA。
  \item \textbf{装饰手势}——wave / point / bow / dance。
\end{itemize}

\begin{secbox}{系统架构}
\normalsize
五层单向数据流：\textbf{输入}$\to$\textbf{解析}（vla/voice/pipeline）$\to$\textbf{控制}
（tasks/control/catcher）$\to$\textbf{物理}（Newton XPBD）$\to$\textbf{渲染}（scene/effects）；
telemetry 横切各层写 CSV。22 模块、约 7730 行。关注点分离使解析层可替换大模型而不影响下游。
\end{secbox}

\vspace{0.5cm}

\begin{secbox}{方法细节}
\normalsize
\begin{itemize}\setlength\itemsep{4pt}
  \item 预飞命中即将程序推入执行器队列，机械臂当帧启动；
  \item Claude 在后台 daemon 线程返回后与预飞比较：相同则静默确认，不同则记录；
  \item 每次派发递增 \texttt{generation} 计数，worker 仅在计数匹配时写回，杜绝过期覆盖。
\end{itemize}
\end{secbox}

\vspace{0.5cm}

\begin{secbox}{性能评估}
\normalsize
\texttt{world.step()} 原占 11.9 ms（71\% 帧预算），主体为 Warp 内核\textbf{启动开销}而非计算。
经降迭代/子步与 teleport 跳过碰撞等优化降至 \textbf{0.30 ms（$\approx$40$\times$）}，uncapped
吞吐 64$\to$\textbf{279 fps}。渲染臂由控制器前向运动学驱动，降迭代对画面无影响。
\end{secbox}

\vspace{0.5cm}

\begin{secbox}{评估与结论}
\normalsize
250 项单元与集成测试（100\% 通过），偏移塔倒塌边界由真求解器回归测试固定。系统在无 GPU、
无云的消费级硬件上实现了\textbf{可解释、低延迟、可复现}的具身智能教学演示。\par
